# Bank Account System - Java

A simple banking system project built with Java.  
This project was created to practice object-oriented programming, user input, conditions, methods and basic banking operations.

## Features

- Create a bank account
- Check account balance
- Deposit money
- Withdraw money
- Validate insufficient balance
- Simple menu-based interaction

## Technologies Used

- Java
- Git
- GitHub
- VS Code

## What I Learned

- How to work with classes and objects in Java
- How to use methods for different actions
- How to use conditions to validate user input
- How to structure a small Java project
- How to upload and manage a project with GitHub
